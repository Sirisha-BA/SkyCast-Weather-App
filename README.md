# SkyCast-Weather-App
A modern weather app with real-time data, animated UI, and dynamic backgrounds using OpenWeatherMap API.

# 🌌 SkyCast — Futuristic Weather Intelligence App

> 🚀 A next-generation weather application with real-time data, dynamic animations, and a premium glassmorphism UI — built for performance, aesthetics, and user experience.

## ✨ Overview

**SkyCast** is a modern web-based weather application that provides **real-time weather insights** using the OpenWeatherMap API.
It combines **clean UI design, animated backgrounds, and interactive components** to deliver an immersive experience beyond traditional weather apps.

---

## 🔥 Key Highlights

* 🌍 **Real-time Weather Data** (Temperature, Humidity, Wind, etc.)
* 🔎 **City-based Search System**
* ⚡ **Fast API Integration (Fetch API)**
* 🎨 **Dynamic Backgrounds based on Weather Conditions**
* 💎 **Glassmorphism UI Design**
* 📅 **5-Day Weather Forecast**
* ⚠️ **Robust Error Handling**
* 📱 **Fully Responsive (Mobile + Desktop)**

---

## 🧠 Problem Statement

Traditional weather apps often lack:

* Engaging UI/UX

```bash

cd skycast-weather-app
```

* Open `index.html`
  OR
* Use **Live Server in VS Code**

## 🔑 API Configuration

1. Get API key from: https://openweathermap.org/
2. Replace in `script.js`:

```javascript
const API_KEY = "YOUR_API_KEY_HERE";
```
